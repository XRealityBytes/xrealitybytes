## Overview

Shader Variant Auditor scans generated shader permutations and flags high-risk branches that can inflate warmup costs or trigger runtime compilation spikes.

## Why It Matters

Renderer teams often discover variant pressure late, when performance budgets are already tight. This tool shifts that feedback left by making variant growth visible during CI.

## Output

Each run produces:

- Variant counts by material family.
- Top conditional features by cardinality impact.
- Recommended merge strategies for reducing branch combinations.
