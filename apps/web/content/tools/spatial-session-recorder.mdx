## Overview

Spatial Session Recorder stores user interactions, camera transforms, and timeline markers in a compact JSON format. It supports deterministic playback so teams can compare behavior across builds and rendering backends.

## Typical Use Cases

- Reproduce subtle interaction bugs in 3D interfaces.
- Create regression clips during renderer refactors.
- Share replayable product walkthroughs with design and QA.

## Key Capabilities

- Event compression for long sessions.
- Timestamp normalization to reduce clock skew impact.
- Branchable playback markers for scenario testing.
